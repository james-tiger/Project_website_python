## Setup & Installation

### Prerequisites

- Python 3.8 or higher
- Virtualenv
- Git installed locally
- MongoDB instance for real-time data
- Access to **hh.ru API** for job vacancy data
- Access to the **Calculation API** for profession-related calculations
